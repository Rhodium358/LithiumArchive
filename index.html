Lithium is a project for building custom Windows ISOs using MSMG Toolkit. It focuses on minimalism, security, and performance. Supports Windows 10 LTSC/IoT and Windows 11 builds.

Features

  • Safe removal of unnecessary components
  • Automated installation of Chrome & LibreOffice
  • Registry tweaks for privacy & performance
  • Persistent unattended setup

Usage

Download the ISO, burn it to USB, and install. All tweaks and programs are applied automatically during setup.
